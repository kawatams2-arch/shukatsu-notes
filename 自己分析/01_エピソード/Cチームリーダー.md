# Cチームリーダー(最重要ガクチカ)

## 📌 確定事実(数字は不変)
- 体育会ラクロス部、A〜CのうちCチーム約25名のリーダー(3年時、推薦で就任)。下級生含む
- 課題:基礎技術と主体性の欠如 → 敗戦が続く
- ボトルネック:**「意図を理解せず漫然と練習をこなす状態」**
- 施策①**自主練の仕組み化**:自主練動画をLINEグループに投稿→進捗可視化→私が個別添削するサイクル(OF×2/DF×1グループに分け、コーチと分担)
- 施策②**練習の自律化**:下級生をメニュー選定に巻き込み(内的動機づけ)、各メニューごとに5分間、意識点をグループで共有
- 成果:**自発的に反省を出すメンバー2名→10名以上** / **ミスによる攻守交代 平均7回→4回(2〜5月)**

## 🧩 因果チェーン(深掘りで崩れない骨格)
敗戦 → 直接原因は基礎技術不足 → 症状は2つ(①漫然としたチーム練習 ②自主練不足) → 共通の根本原因は**主体性の欠如** → 施策は各症状に対応(①練習の自律化 ②自主練の仕組み化)

## ✍️ 常体400字版(汎用・確定)
体育会ラクロス部で、下級生を含むCチーム約25名のリーダーとして組織の底上げに取り組んだ。当初は基礎技術と主体性の欠如で敗戦が続いており、私はボトルネックを「意図を理解せず漫然と練習をこなす状態」による基礎技術不足だと特定した。
そこで2つの施策を実行した。1つ目は自主練の仕組み化。練習動画をLINEに投稿させて進捗を可視化し、私が個別に添削するサイクルを構築した。2つ目は練習の自律化。下級生をメニュー選定に巻き込んで動機づけ、各メニューごとに5分間、意識点をグループで共有する時間を設けた。
結果、ミスによる攻守交代は2〜5月で平均7回から4回に減少し、自発的に反省を出すメンバーも2名から10名以上に増えた。この経験から、課題を構造化し、個と向き合って行動を促す仕組みの重要性を学んだ。

## ✍️ 信頼関係軸400字版(KDDI等・人重視の企業向け)
体育会ラクロス部Cチームのリーダーとして、組織の底上げに挑戦した経験だ。当初チームは敗戦が続き、特に下級生の主体性が低下していた。私はその根本に、指導者との信頼関係の薄さがあると考えた。指導が一方的に感じられ、「やらされている」状態が、自ら考え動く姿勢を奪っていたのだ。
そこで私は、一方的な指導をやめ、「個と徹底的に向き合う姿勢」を大切にした。具体的には、メンバーが自主練動画をLINEに共有し、私が全員に個別で詳細なフィードバックを返す仕組みを作った。さらに、練習メニューの選定に下級生を巻き込み、彼らの意見を現場に反映させた。
この「一人ひとりの成長に寄りそう姿勢」が伝わり、メンバーとの間に強固な信頼が生まれた。結果、自発的に反省を出す部員が2名から10名以上に増え、試合中のミスによる攻守交代も平均7回から4回に減少した。
この経験から、当事者意識を持って相手に向き合い続けることが、人の主体性を引き出し、信頼を築く原動力になると学んだ。

## 🎯 軸の使い分け

- **信頼関係軸**:人重視の企業(KDDI、東京海上、金融、デベロッパー等)
